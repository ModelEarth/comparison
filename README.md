# Comparison

### Modular TradeFlow Comparison Interface

Frontends for tradeflow data with a UN Goal focus. [Related Issue](https://github.com/modelearth/projects/issues/48)

You can fork and contribute updates to existing charts, or experiment here.

1. To experiment, create a folder for your work in comparison/dev using your unique handle (name).
2. Prepare a vibe prompt in your readme and post a link to your page in our Discord.
3. Gather feedback from others and refine your vibe prompt.

[Tradeflow Table Documentation](../exiobase/tradeflow/) - Our source files are output by python in our [exiobase repo](https://github.com/ModelEarth/exiobase/tree/main/tradeflow)
